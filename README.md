[[# News Topic Classifier Using BERT

## Objective
Classify news headlines into 4 categories using BERT transformer model.

## Dataset
- AG News Dataset (120,000 articles)
- Categories: World, Sports, Business, Sci/Tech

## Approach
1. Loaded AG News dataset
2. Tokenized using BERT tokenizer
3. Fine-tuned bert-base-uncased model
4. Trained for 3 epochs

## Results
- **Accuracy:** 93.5%
- **F1-Score:** 0.934

## Technologies
- PyTorch
- Hugging Face Transformers
- Gradio (deployment)
- scikit-learn (evaluation)

```bash

```

